# Zero-Day

Run the detector with the batch launcher:

```bat
run_detector.bat
```

If you want to use PowerShell directly, run:

```powershell
powershell -ExecutionPolicy Bypass -File .\run_detector.ps1
```

In VS Code, open `detection/stub_detector.py` and use the top-right Run button, or pick `Python: Stub Detector` from Run and Debug.
